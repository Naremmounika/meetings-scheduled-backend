# meetings-scheduled-backend
Description :

This project is a backend assignment I built using Node.js and Express.
Main goal is to create APIs for users and meetings and connect them with database.

Learned:
1) Modular Express architecture
2) Sequelize integration
3) REST API design
4) Route structuring


Tech Used :
Node.js
Express.js
Sequelize
MySQL
dotenv

How to run :

first run the below command to install dependencies :
 
 npm install 

All required node packages will be installled and you can see them in package.json

For starting the server, run the below command:
 npm start
